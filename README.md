# What the project is about
A system that stores, processes and analyses mock examination data for individual department. This project brings together python projects so far and organises them into a practical and realistic workflow.

# Usage
```cmd
cd C:\User\(your username)
git clone https://github.com/Timiz1/gse301-assignment-submission.git
cd gse301-assignment-submission
python student-management.py
```
